## 游戏开发聚会 GDM（Game Development Meetup）

对游戏开发感兴趣的人都可以参加。

当然，不限国籍。我们希望这是一个全世界的人都能聚集的会议。

关于内容，我们没有特别的设定，所以希望它能作为一个自由交流游戏开发的场所。

---

隔周星期日，20:00～21:00
